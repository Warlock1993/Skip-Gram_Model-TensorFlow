# Skip-Gram-Model-TensorFlow
TensorFlow implementation of the word2vec (skip-gram model)


# Requirements
* tensorflow >= 2.0    
* numpy >= 1.18      
* matplotlib       
* tqdm 
* nltk
* gensim


# Training
```
python main.py
```

# Testing
```
python test.py
```
